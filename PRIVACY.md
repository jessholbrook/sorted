# Sorted Privacy Policy

Effective date: June 13, 2026

Sorted does not collect, store, sell, or transmit personal data.

## Accessibility Access

Sorted requests macOS Accessibility permission so it can:

- Read the titles, minimized state, and positions of open windows.
- Identify minimized-window thumbnails in the Dock.
- Move and resize windows when the user selects an arrangement action.
- Perform pointer drag events when the user requests live Dock sorting.

This information is processed locally on the Mac only. Sorted does not retain
window titles or positions after completing an action.

## Network Access

Sorted does not make network requests and does not include analytics,
advertising, crash-reporting, or tracking SDKs.

## Data Sharing

Sorted does not share data with the developer or any third party.

## Changes

Material changes to this policy will be documented in the project repository
and reflected by an updated effective date.
